NFL RPG GAME

The objective of our project was to create a functional RPG Game. First the Users selects a player from a group of characters. When the User selects a player, the rest of the characters become opponents. Then the User will use its character to attack the opponents one at a time until all of their health points reach zero.

In this project HTML was used to create a WebPage and then it was styled with CSS. Then we created the functions to play the game with Javascript and JQuery.

Another add on to the Project was the creation of the reset button that would allow the user to replay the game without having to refresh the page.

Visit the NFL RPG Game to give it a try.

SPRUS TRIVIA GAME

The objective of our Trivia Project was to create a timed Questionaire. The User would have a certain amount of time to answer question about a topic. At the end of the Trivia Game they would see how many questions they got Right or Wrong.

In this project the usual HTML, CSS, Javascript and JQuery were used to make the page. Then using setTimeout and setInterval to work the countdown for the game.

Visit the San Antonio Spur's Trivia Game to give it a shot.

TRAIN ARRIVAL TIMES

The objective of the Train Arrival Project was to make a chart of train arrivals based on what the User inputed. The chart would automatically calculate the next time the train would be arriving to the station. Concepts used for the project were Moment.js that was used to calculate the time, and Firbase, an online data storage site.

Technologies Used: HTML, CSS, Javascript, JQuery, Firebase and Moment.js.

Please Check out Train Arrival Times to see when the next train arrives.
